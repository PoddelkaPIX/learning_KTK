Запуск осуществляется через файл main.go который находится по пути backend/main.go относительно корня проекта.

Для запуска на компьютере должно быть установлено:
	Go - go1.17.5
	PostgreSQL - 14+

Я приложил готовую базу данных. Чтобы её загрузить, нужно создать базу данных с именем "learning" на порту 5432 
у стандартного пользователя postgres через pgAdmin. Далее скопировать SQL код из файла dataBase.txt 
и вставить в запросник у базы данных "learning".

Вся настройка для запуска сайта на компьютере происходит в файле settings.cfg в корне проекта.
В нём нужно указать свои параметры базы данных если у вас они отличаются от тех, что уже указаны в файле.
